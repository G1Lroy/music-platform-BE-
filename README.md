## Github

```bash
$ git clone https://github.com/G1Lroy/music-platform-BE-.git
```


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
